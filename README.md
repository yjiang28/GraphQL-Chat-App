## A full stack chatting app

Frontend: Next.js, React, Apollo Client, Material-UI  
Backend: Node.js, Apollo Server, Prisma db

Features:

-   Sign up, sign in, sign out
-   Search for users
-   Add friend
-   Pubsub notification
-   Real-time messaging
